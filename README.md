# Easy Common
A set of useful utilities and helpers used across Easy.* projects.

Supports _.NET Core_ (_.NET 4.5_ & _netstandard2.0_) running on:
* .NET Core
* .NET Framework 4.5 and above
* Mono 5.4
* Xamarin.iOS 10.14
* Xamarin.Mac 3.8
* Xamarin.Android 7.5
___
